How enforcement works

Tiered roles

Super Admin ⊃ Admin ⊃ Member. Each tier inherits everything below it. Pages check requiredRoles.includes(role); APIs use requireRole([...]).

Guest allowlist

Guests inherit nothing — deny by default. Access comes from rows in guest_permissions (one per page, with can_view / can_edit / can_delete).

Guest redirects

A guest hitting a page not on their allowlist is redirected to their first permitted page. The sidebar hides everything they can't view.

Who assigns roles

Only Super Admins change roles or grant Super Admin. Admins can approve join requests and configure each guest's per-page permissions.
